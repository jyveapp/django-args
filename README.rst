django-args
########################################################################

Documentation
=============

`View the django-args docs here
<https://django-args.readthedocs.io/>`_.

Installation
============

Install django-args with::

    pip3 install django-args

After this, add ``djarg`` to the ``INSTALLED_APPS``
setting of your Django project.
